=== Lollum Page Builder ===
Requires at least: 4.1
Tested up to: 4.7
License: GPLv3

A simple and lightweight page builder for developers.

== Description ==

The Lollum Page Builder plugin is a simple page builder meant for developers. It doesn't provide tons of features, mega-effects or super options. It's a simple grid with a few predefined blocks. And it provides a simple filter for the creation of new blocks.

== Frequently Asked Questions ==

= Can I use this plugin with other themes, premium or free? =

Yes.
